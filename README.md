## 📋 Overview
A powerful **Visual Retrieval-Augmented Generation (Visual RAG)** application that extracts text from images using OCR and answers questions intelligently using vector embeddings.

## ✨ Features
- OCR using Tesseract
- Semantic Search with Sentence Transformers
- Vector Database (PostgreSQL + pgvector)
- Powered by OpenRouter + GPT-3.5
- Beautiful Modern UI
