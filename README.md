# 📖 About the Project

**VisualRAG** is an AI-powered document intelligence system that allows users to upload document images, extract text using OCR, store the extracted information as vector embeddings, and ask questions in natural language.

Instead of simply extracting text, the application understands the document's context using Retrieval-Augmented Generation (RAG), making document search and question answering much smarter.

---

# ✨ Features

✅ Upload one or multiple document images

✅ OCR-based text extraction using **Tesseract OCR**

✅ Semantic search with **Sentence Transformers**

✅ Vector storage using **PostgreSQL + pgvector**

✅ AI-powered question answering using **OpenRouter LLM**

✅ Clean and interactive **Streamlit UI**

✅ Fast contextual retrieval using embeddings

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/VisualRAG.git
```

Move into the project folder

```bash
cd VisualRAG
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
